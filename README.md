# Student Health Tracker

## Description
This is a CLI-based application that helps students track their daily health habits such as food intake, water consumption, and bowel movements.

## Features
- Add daily health logs
- View previous records
- Basic health analysis (hydration & constipation detection)

## Technologies Used
- Python
- JSON (for storage)

## Installation
1. Clone the repository:
   https://github.com/anushka24bai10102-coder/health-tracker

2. Navigate to project folder:
   cd health-tracker

3. Run the program:
   python main.py

## Usage
- Choose options from menu
- Enter daily health data
- View logs and analysis
